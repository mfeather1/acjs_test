1000x1000 Cube
Test Index



This version supports cube sizes up to 1000x1000 with SiGN notation, only single layer slice moves are supported. How large of a cube that can be animated depends on the processing capability of your computer. Test a move sequence by clicking on a link below. To test cube sizes other than the ones shown change the dim (dimension) parameter in the url after clicking on one of the links.

To maximize the cube size for which animation can be seen rotate the cube so only one face is showing prior to starting the sequence, also press the borders button to eliminate drawing of the black background between facelets. With larger cube sizes the number of frames of animation per move will decline but at least one frame of animation is visible until well over 500x500 (slight movement is still visible at 600x600) with the Ryzen 5 Quad-Core as a benchmark. In this test, Firefox significantly outperforms Chrome which is the opposite result compared to the Optimal Solver benchmark tests in which Chrome has a considerable performance advantage.
