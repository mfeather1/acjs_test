Responsive Cubes

This page is an example of how cubes can be resized both vertically and horizontally based on a percentage of the window width. These cubes are set at 20 and 25 percent respectively (decimal numbers are also allowed for more exact sizing). The resizing of first 4 cubes is done by a small bit of JavaScript, see source code of this page for details. There is a minimum width defined for window size which is 400, so the first cube would always be at least 80 pixels wide.



Next 2 cubes are a little larger because the height is increased to adjust for the button bar and slider (see best appearance recommendation in the Using Simulator in Web Pages section in AnimCubeJS documentation).



The resizing of next 4 cubes is done by a small bit of CSS (without using JavaScript), see source code of this page for details. There is a minimum width defined for first cube which is 80 pixels.



Next 2 cubes are a little larger because the height is increased to adjust for the button bar and slider (see best appearance recommendation in the Using Simulator in Web Pages section in AnimCubeJS documentation).
