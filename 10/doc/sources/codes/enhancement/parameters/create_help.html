Customizing AnimCubeJS
Help

Start by selecting a cube size from the drop-down menu, then declare parameters in the text box next to the simulator.


The following is an example of how to enter parameters in a user-friendly way, use an & prefix for all but the first one:


move=R U' L F U' R2 U2 R U R' U2 D' L D F2 L2 U
&initrevmove=#
&colorscheme=ROYWGB
&movetext=1
&textsize=18
&buttonheight=18
&bgcolor=dddddd
&butbgcolor=baffc9
&fonttype=0
&align=0
&scale=1

See AnimCubeJS documentation for the complete list of parameters.


If you press the Apply button then:

  1. Specified parameters from the text entry box will be applied to the simulator.

  2. Entered parameters will be displayed in a computer-friendly form in the non-editable text field below the buttons.

  3. The link to your simulator will be generated. You can directly modify generated URL query string to change the simulator functionality.

By pressing the Clear button both text fields will be cleared, and the URL link will be removed. The simulator will also be initialized to its default configuration.
